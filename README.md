# homebrew-go-playball

Homebrew tap for [go-playball](https://github.com/pdavlin/go-playball) — a terminal-based MLB game viewer.

## Install

```bash
brew install pdavlin/go-playball/go-playball
```

## Update

```bash
brew upgrade go-playball
```

This formula is automatically updated by CI when a new release is published.
